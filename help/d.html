Exporting results

Export to XLSX, ODS, CSV
To export the data in the result table to any of the mentioned formats click on the "Export" button under the result table. In the file save dialog chose your wanted save location, file name and file type. The latter is possible by clicking on the triangle of the dropdown menu at the very bottom of the window.

Export to BPC
To export the data in the result table to BPC click on the "Export to BPC" button under the result table. In the file save dialog chose your wanted save location and file name.

Export to EDDN
To export the results to Elite Dangerous Data Network click on the "Export to EDDN" button. This will upload you result table to EDDN. The progress is indicated in the the status bar in the bottom of the window.

Trade Dangerous Export
By clicking "Trade Dangerous Export" you will create a "import.prices" file with the result data in your standard export location. You can find this location in Settings. Trade Dangerous Export is a plugin written by gazelle(on frontier dev forum). You can remove this button by deleting TD_Export directory in the plugins directory.

Limitations
There are no limitations for export of the data for own usage (switch off public mode). However export to BPC and EDDN is limited to following conditions:
- All results must contain system name
- Results cannot be older than 2 hours
