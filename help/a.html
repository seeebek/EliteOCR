Adding Files

To open all files from your screenshot directory click on "Add all". To add single files for OCR click on the green + on the left side of the window. Alternativelly you can click on File in the menu bar an chose "Open".

The following window allows you to chose files which will be added to file list. You can chose more than one file by holding CTRL while clicking on the files.

Files added to file list will be processed to find the fitting system name in log files. This might take a little longer if you have many (and long) log files which fit the creation date of the screenshot.

Upon selecting a file in the list the image will be read from your hard drive and processed by auto-recognition and auto-calibration.

You can remove unwanted files from the list by selecting it and clicking on -. You can clear the file list by clicking on "Remove All".
